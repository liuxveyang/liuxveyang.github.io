codeforces hungry sequence 水题

July 19, 2013
CodeForces

题目链接：http://codeforces.com/problemset/problem/327/B 这道题目虽然超级简单，但是当初我还真的没有想出来做法，囧，看完别人的代码恍然大悟。


#include <cstdio>
#include <cstdlib>
#include <cmath>
int main(void) {
    int n; scanf("%d", &n);
    for (int i = n+1; i <= n+n; ++i) {
        printf("%d", i);
        if (i != n+n) printf(" ");
    }
    printf("\n");

    return 0;
}

＝＝只需要注意到整除这个条件：至少是2倍，所以只需要从n+1一直到2*n就可以了，正好n个数字，最大的数字和最小的数字相比也不超过2.就是这么个简单的道理。
