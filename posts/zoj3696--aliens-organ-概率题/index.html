题目链接：http://acm.zju.edu.cn/onlinejudge/showProblem.do?problemCode=3696 题目大意： 　　就是一个东西，随机产生器官，一天产生的数量的期望是x,求一天内产生个数小于等于N的概率 题目思路： 　　泊松分布。概率论得好好学。

  {1,1},{1,-1},{-1,-1}};
double la;
double cal(double k) {
  double l = 1; 
  for (int i = 1; i <= k; ++i)  {
    l = l * i;
  }
  double re = pow(la, k) / l * exp(-la);
  return re;
}

      sum += cal(i*1.0);
    }
    printf("%.3lf\n", sum);
  }

  return 0;
}
