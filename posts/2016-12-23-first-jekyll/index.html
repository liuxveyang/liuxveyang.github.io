第一次使用Jekyll来搭建博客

December 23, 2016
jekyll

第一次使用Jekyll来搭建静态博客。这里有一份关于Markdown语法的文档：Markdown，挺不错的。

如果想要在Jekyll的Markdown里面使用Github图标的形式插入Github的项目地址，可以这样：

{% highlight markdown %} {\% include icon-github.html username=“username” %} / repository-name {% endhighlight %}

注意：需要把上面开头的两个\去掉。

这个博客的源地址在这里：{% include icon-github.html username=“replper” %} / replper.github.io
